# showdoc开源文档系统的搭建

提示

在阅读此文章前默认您已经阅读过快速上手这篇文章

ShowDoc官网：showdocopen in new window

ShowDoc是一个适合IT团队的在线API(接口)文档、技术文档的编辑工具，通过showdoc你可以方便地使用Markdown语法来书写出美观的API文档、数据字典文档、技术文档、在线excel文档等等

演示网站open in new window

# 源码下载并上传服务器

从Github自动获取git clone https://github.com/star7th/showdoc.git

或手动下载showdocopen in new window

或点击此处open in new window直接下载

然后上传服务器并解压，将showdoc文件夹下的所有文件放置在根目录下或进行重定向

# 安装

进入自己的域名，然后选择语言（English/简体中文）

默认账号 用户名：showdoc 密码：123456

再次刷新页面，选择我的项目，进入设置

对管理员账户进行设置

再到站点设置中

网站外部访问url需要进行填写

安装就到此结束了，你可以回到“我的项目”中创建项目并开始你的编写
